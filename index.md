# UNL Navigation SDK for Android
The UNL Navigation SDK for Android suite provides a toolkit for building navigation-enabled applications, including global map display, route calculation, turn-by-turn navigation, real-time traffic data, and offline navigation.

The SDK is designed for performance and flexibility, giving developers control over application architecture and user experience while integrating navigation capabilities.
