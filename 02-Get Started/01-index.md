# Get Started

With these resources, you will be able to create advanced navigation and mapping applications efficiently.

   - This guide describes the steps to install the most recent version of the Navigation SDK for Android and configure your Android app to use the SDK.
- [Create Your First App](/02-Get%20Started/03-Create%20Your%20First%20App.md)
   - This guide describes the steps to create a new Android application that uses the Navigation SDK for Android.
