# Display map items

This collection of articles covers a wide range of features and techniques for displaying various elements on a map within a mobile application, including landmarks, markers, overlays, routes, instructions, and paths.

- [Display Landmarks](/04-Maps/05-Display%20Map%20Items/02-Display%20Landmarks.md)
   - When displaying the map, we can choose what types of landmarks we want to display. Each landmark can have one or more UnlLandmarkCategory.
- [Display Markers](/04-Maps/05-Display%20Map%20Items/03-Display%20Markers.md)
   - The base class for the marker hierarchy is UnlMarker. It encapsulates coordinates assigned to a specific part.
- [Display Overlays](/04-Maps/05-Display%20Map%20Items/04-Display%20Overlays.md)
   - Overlays are used to provide enhanced, layered information that adds contextual value to a base map, offering users dynamic, interactive, and customized data that can be visualized on top of other map elements.
- [Display Routes](/04-Maps/05-Display%20Map%20Items/05-Display%20Routes.md)
   - Single or multiple routes can be displayed on the map, with only one designated as the main route and the others treated as secondary.
- [Display Route Instructions](/04-Maps/05-Display%20Map%20Items/06-Display%20Route%20Instructions.md)
   - Instructions are represented as arrows on the map.
- [Display Paths](/04-Maps/05-Display%20Map%20Items/08-Display%20Paths.md)
   - Paths can be displayed on the map by adding them into the path collection.
